# Primera Plana

This is an open-source coding philosophy project. 

## Domain for examples

All code examples use the "Tostado" domain — a coffee roastery e-commerce platform:
- Orders, subscriptions, roast batches, shipments
- Partners: coffee farmers, logistics providers
- Inventory management, quality control

Never use N26, investments, banking, or any company-specific domain in examples.

## Writing style

- Write with conviction but respect for alternatives
- Show before/after examples for every rule
- The reader should feel the pain of "before" viscerally
- The "after" should feel like a relief
- Be honest: this philosophy has tradeoffs (more methods, more indirection) — acknowledge them

## Languages

- Philosophy is language-agnostic
- Kotlin is the primary reference implementation
- Swift and TypeScript are secondary guides
- All three should feel native to their ecosystem, not like ports
